# Taskferry Brand and Mascot Design

## Goal

Create a distinctive Taskferry logo and mascot for an MCP server that reliably routes
and monitors background agent work. The brand must feel like a clever technical
courier, not a generic chatbot or a children's character.

## Core Direction

The mascot is a streamlined otter messenger in three-quarter profile. It carries a
small task canister or dispatch pennant. A single route curve, formed by its tail or
wake, signals controlled movement between agent tasks.

## Visual System

- Use compact, flat vector geometry with a bold uniform outline.
- Make the mascot recognizable at favicon scale inside a circle or rounded square.
- Use a horizontal lockup with the mascot mark and lowercase `taskferry` wordmark.
- Use near-black navy, electric lime, ferry orange, and off-white.
- Avoid gradients, photorealism, robot imagery, and overly cute facial features.
- Keep the silhouette legible on light and dark backgrounds.

## Lavish Exploration

Create a local HTML concept board with three variations:

1. **Signal Otter:** carries a dispatch pennant and uses a route arc as its primary
   graphic element.
2. **Wake Otter:** moves through a compact wake that creates the Taskferry mark.
3. **Cargo Otter:** carries a small sealed task canister to emphasize dependable
   delivery.

Each variation must show a favicon, horizontal lockup, light-background application,
and dark-background application. The board will use Lavish annotations to select and
refine the winning direction.

## Final Asset

After review, generate the winning design through Recraft as an SVG. Inspect the SVG
at icon and lockup sizes before accepting it. Deliver the final SVG and any useful
PNG preview beside it in a dedicated brand-assets directory.
